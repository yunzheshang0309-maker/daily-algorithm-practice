# 每日算法+AI手撕练习

> **"任何值得做的事，做得糟糕也值得做。"** —— J. S. 密尔

---

## 简介

这是一个交互式的每日练习脚本，专为正在准备算法面试和AI面试的同学设计。

### 核心功能

| 功能                   | 说明                         |
| ---------------------- | ---------------------------- |
| 每日2道LeetCode Hot100 | 随机抽取，从Easy到Medium     |
| 每日1个LLM知识点       | 覆盖Transformer、RLHF、RAG等 |
| 详细讲解               | 思路分析 + 代码 + 常见坑点   |
| 挖空练习               | 随机挖空2处，检验理解        |
| 完整手写               | 最后自己独立写一遍           |

---

## 环境要求

- Python 3.7+
- 无需额外依赖包（只使用标准库）

## 快速开始

### 1. 目录结构

```
D:/2026_project/
├── daily_practice.py      # 主脚本
├── Hot100_Easy.md         # LeetCode简单题库
├── Hot100_Medium.md       # LeetCode中等题库
└── daily_progress.json    # 学习进度（自动生成）
```

### 2. 运行

```bash
python D:/2026_project/daily_practice.py
```

### 3. 交互流程

```
***************************
**  每日算法+AI手撕练习  **
***************************

[OK] 加载了 2 道题目
```

每道题会依次展示：

1. 题目描述 + 核心思路 + 标准代码
2. 挖空练习（2处）
3. 完整手写

## 学习进度

脚本会自动记录你的学习进度，保存到 `daily_progress.json`：

```json
{
  "leetcode_done": ["lc_1", "lc_206", ...],
  "llm_done": ["Transformer基本结构", "LoRA微调原理"],
  "last_date": "2026-03-10"

```

- 已完成的题目不会重复出现
- 每天首次运行会加载新题目
- 同一题目做完后会自动记录

### 零基础入门（前2周）

| 优先级 | 题型   | 推荐题目                         |
| ------ | ------ | -------------------------------- |

## 常见问题

### Q: 题目加载失败怎么办？

确保 `Hot100_Easy.md` 和 `Hot100_Medium.md` 在同一目录下。

### Q: 挖空答案是随机生成的，可能不太合理？

挖空逻辑会尽量选取代码中的关键行（如赋值、循环条件等）。如果遇到不合理的挖空，可以直接看答案理解。

### Q: 写不出来怎么办？

这是正常现象！建议的学习方法：

1. **第一遍**：先看答案，理解思路
2. **第二遍**：抄写代码3遍，形成肌肉记忆
3. **第三遍**：自己独立写
4. **第四遍**：隔天再写一遍巩固

### Q: 进度可以重置吗？

可以。删除 `daily_progress.json` 即可重置进度。

## 进阶用法

### 每日自动运行（Windows任务计划）

1. 创建批处理文件 `run_practice.bat`：

```batch
@echo off
python D:\2026_project\daily_practice.py
pause
```

2. 设置任务计划程序，每天固定时间提醒

### 抓取最新题目

如果想更新LeetCode题目，可以重新下载：

```bash
python -c "import urllib.request; open('Hot100_Easy.md','w',encoding='utf-8').write(urllib.request.urlopen('https://raw.githubusercontent.com/realnghon/LeetCode_Hot100_Python/main/Hot100_Easy.md').read().decode('utf-8'))"
```
